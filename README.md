lightweight, blazing fast tag-based file manager

### features
+ entirely written in rust (except for rusqlite)
+ advanced file relationships using sqlite
+ quick media preview
+ integration with external apps for customized media CRUD

### to run: 

```bash
# build the project
cargo build

#run the app
cargo run --release
```
